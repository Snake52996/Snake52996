地图工坊作品-安娜瞬镜一枪

发表于 2019-07-10 | 分类于 守望先锋地图工坊 | |

前言

安娜练枪模式。
据说来这里疯狂挨打一段时间水平就会提升，甚至最后能吊打别人也说不定(挨打好快乐啊)
限制了只能瞬镜，添加了一些蜜汁特效和彩蛋。
原作者为JINKO，rig搬运(2019/5/23)，我进行了一些（自认为）提升了游戏性的改进

说明

  • 单次开镜不得超过0.6s，以按下开镜按键计时，超过将被打断（眩晕0.4s）并受到10点伤害。
  • 分数达到30将给予火焰特效
  • 分数达到34将给予激素效果，同时禁用武器（只能使用麻醉镖），此时再击杀一人即可获胜

具体设置

  • 描述：瞬镜,开镜不得超过0.6s,击杀30人给予火焰特效，34击杀给予激素，且只能通过睡针击杀。原作者、搬运者等详情参见https://snake.moe/
  • 大厅
    • 地图轮换：完成比赛后
    • 返回大厅：从不
  • 模式：仅开启死斗
    • 允许切换英雄：关闭
    • 复活时间：0%
    • 急救包刷新：禁用
    • 敌方生命条：关闭
    • 比赛时常（分钟）：15
    • 比赛模式开始：即时
    • 消灭镜头：关闭
    • 获胜得分：35
  • 地图
    • 城堡
    • 艾兴瓦尔德
  • 英雄：仅允许安娜
    • 伤害量：500%
    • 受到伤害量：500%
    • 无需装弹：开启
    • 生命值：10%
    • 生物手雷：关闭
    • 禁止自动射击：开启
    • 近身攻击：关闭
    • 麻醉镖 冷却时间：21%
    • 终极技能持续时间无限：开启

地图工坊代码全选并复制文本后即可粘贴进游戏中的地图工坊编辑器现在粘贴过程中由于守望先锋自身的bug原因不能顺利进行，请自行寻找解决办法，如手动输入、删除提示有问题的行粘贴后再手动添加（官方bug无能为力）。其他设置请参照上方的“具体设置”栏目。
